# Deutsch.css

This is a simple CSS snippet to translate any website into **Deutsch** :-)

import { useState } from 'react'

export function Preview({ code }) {
  const [enabled, toggle] = useState(false)
  const A = 'a'
  return <div onClick={() => toggle(!enabled)} style={{ display: 'inline-block', cursor: 'pointer' }}>
    <style dangerouslySetInnerHTML={{__html: enabled ? code : ''}}
    />
    <A href='#'>{enabled ? 'Disable it' : 'Try it on this page'}</A>.
  </div>
}

Similarily, you can use `text-transform` and `::first-letter`, together with the `uncode-range` / `word-spacing` tricks to display a paragraph in camelCase / PascalCase.
